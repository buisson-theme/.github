# Buisson

Botanical inks, warm paper, and contrasts calibrated to the pixel.

Buisson was born from a simple frustration: Evergarden was beautiful,
Flexoki was accessible, and nothing was both.

## Palette

6 botanical accents — hibiscus · sage · river moss · slate sky · thistle · ochre

Each color pair calibrated against WCAG 2.1. Main text at 11.55:1 AAA.

[Design tokens](https://github.com/buisson-theme/buisson/blob/main/tokens/buisson-tokens.json)
